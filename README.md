A premium, full-stack medical diagnostic web application that combines computer vision, advanced large language models, and text-to-speech capabilities to simulate an intelligent clinical assistant. Built with a modern, glassmorphic UI, it processes patient images and queries to provide structured diagnostic dossiers and vocalized analysis.

> **Disclaimer:** This is a portfolio project and technical demonstration. It is not intended for actual medical diagnosis. Always consult a certified dermatologist or medical professional for real health concerns.

---

## ✨ Key Features

* **Multi-Modal AI Analysis:** Upload medical imagery alongside text queries for comprehensive visual and contextual analysis.
* **Premium Clinical UI:** A highly unique, modern diagnostic dashboard built with React, Tailwind CSS, and Framer Motion, featuring dynamic scanning effects and a formal "Medical Dossier" layout.
* **Voice Synthesis Engine:** Integrates ElevenLabs API to automatically vocalize the AI's diagnostic findings with ultra-realistic human speech.
* **Real-time Memory Sync:** Maintains conversational context and clinical history across the entire session.
* **High-Performance Backend:** Powered by a lightweight, asynchronous Python FastAPI server to handle image routing and API orchestration seamlessly.

---

## 🛠️ Tech Stack

**Frontend:**
* React.js
* Tailwind CSS (Styling)
* Framer Motion (Animations)
* HTML5 Audio API

**Backend & AI:**
* Python 3.x
* FastAPI
* Uvicorn
* Groq API (Vision & LLM Processing)
* ElevenLabs API (Text-to-Speech)
